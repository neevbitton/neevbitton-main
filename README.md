# How to setup the project

### Step 01
Download and extract the project


### Step 02
To install all dependancies run this command on your terminal
```
npm insall
```

### Step 03
Run this command
```
npx prisma db push
```

### Step 04
Run this commad for creating first admin
```
npm run seed
```

### Step 05
Run this for start application
```
npm start
```
